# Install requirements:
    - Pyqt5
    - Torch
    - Torchvision
# Elements:
    - main.py: file dùng để lập trình sự kiện giao diện
    - Giaodien.ui, Giaodien.py: file giao diện được tạo ra từ Qtdesigner
    - weight0.pth: file trọng số của model DL    
# How to convert ui to py: 
    - Use command: pyuic5 -x Giaodien.ui -o Giaodien.py    
# How to run program:
    - Use command: python main.py
